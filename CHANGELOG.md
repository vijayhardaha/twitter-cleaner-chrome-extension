## [1.0.0] - 2024-05-18

### Added

- Initial release of Twitter Cleaner Chrome extension.
- Features: Delete tweets, delete replies, unlike all likes, remove reposts, live stats, stop button.
- Support for one‑click cleanup with progress display.
